#About the Project
Book Roulette is a fun and intuitive application designed to help readers break out of reading slumps. Instead of overthinking what to read next, users spin the roulette and get a randomly selected book recommendation based on available data or preferences. The idea is to make reading spontaneous, exciting, and stress‑free.
This project blends logic, randomness, and a love for books—turning decision fatigue into a delightful experience.
